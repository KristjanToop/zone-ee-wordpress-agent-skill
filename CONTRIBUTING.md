# Contributing

Keep this repository suitable for public reuse.

- Add only generic, reusable workflows.
- Do not add client requests, direct task prompts, real site URLs, credentials, backups, account screenshots or private operating details.
- Preserve staging-first and explicit-production-approval safeguards.
- Keep the `SKILL.md` frontmatter compatible with the Agent Skills specification.
- Update the version in `SKILL.md` metadata when changing the workflow materially.
